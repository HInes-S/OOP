Функциональные возможности:

- Создание фигуры
	
	Пользователь вводит команду draw <figure> <Name> <x> <y> <w> <h> <BorderColor> <BackgroundColor> <Layer>,
	после чего на холсте рисуется фигура figure(эллипс(ellipse) или прямоугольник(rectangle))
	с заданной позицией (x, y), размером(w, h), цветом границы(BorderColor, где цвет - это символ, пробел (space) или прозрачный (empty)), цветом фона(BackgroundColor), и слоем
	
- Очистка холста

	Пользователь вводит команду clear, после чего с холста удаляются все фигуры
	
- Сохранение холста

	Пользователь вводит команду save <filepath>, после чего текущий холст сохраняется в файле filepath
	
- Загрузка холста

	Пользователь вводит команду load <filepath>, после чего из файла filepath загружается холст
	
- Изменение цвета фона холста

	Пользователь вводит команду canvas color <Color>, после чего фон холста меняет цвет на Color(где цвет - это символ или пробел(space))
	
- Изменение размера холста

	Пользователь вводит команду canvas size <w> <h>, после чего размер холста становится (w, h), где w - ширина, h - высота холста в символах консоли
	
- Удаление фигуры

	Пользователь вводит команду delete <Name>, после чего фигура с именем Name удаляется
	
- Перемещение фигуры

	Пользователь вводит команду figure <Name> pos <x> <y>, после чего фигура с именем Name перемещается на точку (x, y)
	
- Изменение цвета фигуры

	Пользователь вводит команду figure <Name> color <BorderColor> <BackgroundColor>, после чего цвет границы фигуры с именем Name меняется на BorderColor,
	а цвет фона - на BackgroundColor, где цвет - это символ, пробел (space) или прозрачный (empty)
	
- Изменение слоя фигуры

	Пользователь вводит команду figure <Name> layer <Layer>, после чего слой фигуры с именем Name меняется на Layer
	
- Изменение размера фигуры

	Пользователь вводит команду figure <Name> size <w> <h>, после чего размер фигуры с именем Name меняется на (w, h), где w - ширина, h - высота фигуры в символах консоли
	
- Просмотр свойств фигуры

	Пользователь вводит команду figure <Name>, после чего в консоль выводятся свойства фигуры с именем Name: позиция, размер, слой, цвет границы, цвет фона, тип фигуры
	
- Просмотр имён всех фигура

	Пользователь вводит команду figure names, после чего в консоль выводятся имена всех фигур, находящихся на холсте
	
- Отмена изменения

	Пользователь вводит команду undo, после чего отменяется последнее изменение, отмена работает до 10 раз
	
- Повтор изменения

	Пользователь вводит команду redo, после чего восстанавливается последнее изменения, повтор работает до 10 раз
	
- Завершение работы

	Пользователь вводит команду exit, после чего программа завершает работу
	
- Доступные команды

	Пользователь вводит команду help, после чего в консоль выводятся имена всех доступных команду
	
- Подсказка по команде

	Пользователь вводит команду help <command>, после чего в консоль выводится подсказка по команде command
